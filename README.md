# QDApp Template

A starter template for building Qubic DApp.

## Features

- Modern project structure
- Easy setup and configuration
- Example components and utilities
- Wallet connect

## Getting Started

1. Clone this repository:
   ```
   git clone https://github.com/serendipity-seeker/qdapp-template.git
   ```
2. Install dependencies:
   ```
   cd qdapp-template
   npm install
   ```
3. Start the development server:
   ```
   npm start
   ```
